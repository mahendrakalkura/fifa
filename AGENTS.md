# AGENTS.md

## Project

Fish script that displays the FIFA World Cup 2026 schedule in a terminal ASCII table, grouped by day, with the next upcoming game highlighted.

## Key files

- `fifa.fish` - standalone fish script
- `fifa.json` - match data

## Dependencies

- fish shell
- jq
- awk
- date
- tput (interactive view only)
- less (interactive view only)

## Conventions

- No comments in code
- Alphabetical ordering
- ASCII table format with `+`, `-`, `|` characters
